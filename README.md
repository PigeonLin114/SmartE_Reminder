珠海科技学院宿舍电费缴纳提醒程序

## 1.功能实现

1.自动抓取“我的珠科”APP中“智能水电”的数据

2.自动推送剩余电量信息


## 2.功能模块划分

1.抓包模块

抓取APP中“智能水电”数据并转化为JSON格式

~~2.判断模块（废除）~~

~~读取JSON数据中的“剩余电量信息”，当低于10度时拉起消息推送模块~~

2.消息推送模块

每日通过邮件推送电量信息

3.定时模块

每日定时启动程序（由Windows任务计划承担）


## 3.项目说明

### 3.1抓包部分

在抓包中，我们需要用到Reqable（小黄鸟）和安卓模拟器（须启用root，方便部署系统证书）对“我的珠科”APP中的“智能水电”部分进行抓包，根据软件提示进行连接并部署SSL自签名证书（必须，否则无法正常使用“我的珠科”），进行抓包后我们可以得到请求头和cookie ，请求体（请求头已经在程序中写好，唯一要关注的就是cookie和请求体）获取到这些信息后填入程序JSON_GET.py中


### 3.2消息推送部分

在EmailSend.py中，按照注释填写收件，发件邮箱等信息


### 3.3定时部分

Windows计划任务解君愁（建议对项目进行打包成EXE文件
